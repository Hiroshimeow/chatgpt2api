# Review

Checklist review trước khi merge/push:

1. Kiểm tra remote và branch.
2. Kiểm tra dirty files, loại trừ dữ liệu local.
3. Đọc `todo.md`, `plan.md`, `done.md`.
4. Đảm bảo thay đổi đúng scope.
5. Chạy compile/test tối thiểu.
6. Ghi rõ phần đã kiểm chứng và phần chưa kiểm chứng.

Không rubber-stamp. Nếu có rủi ro, ghi thẳng rủi ro.
